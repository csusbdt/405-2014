Overview

The purpose of this assignment is to learn a little about relational databases and how to use a relational database called PostqreSQL.

How to pronounce PostgreSQL?

This assignment requires research and self-study. If you get to this assignment early enough, I can help you solve some of the problems you encounter.

Find and do a tutorial to learn the basics of SQL using the PostgreSQL database. The tutorial in the PostgreSQL documentation may be sufficient.

Assignment Folder

Create a directory named psql for the work you do for this assignment. Start this assignment by making a copy of your solution to the gift assignment.

Reading

You will need to become familiar with the SQL language and the PostgreSQL documentation. Don't get absorbed with learning the advanced features of SQL; the following are the SQL operations I believe you will need to use.

  • create a database
  • delete a database
  • create a table
  • insert a new row in a table
  • get a row of data using a primary key
  • update a row in a table based on a primary key

Instructions

Install PostgreSQL and start it.

There are 2 ways you should interact with the database:

  • From the command line client psql.
  • Through a PostreSQL driver written for Node.js.

You should become familiar with the command line client because you will use it for debugging and for writing a database creation script. You should write a script that creates a database called gift and populate it with the same data used to test the code you developed for the gift application.

Unlike CouchDB, you'll need to use a driver written for Node.js in order to interact with the PostgreSQL database. You should use the client pooling feature of the driver rather than open and close a connection with each access to the database.

To complete this assignment, I believe that you only need to modify the database creation script and the db module defined in db.js.
